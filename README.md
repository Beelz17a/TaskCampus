# TaskCampus

TaskCampus es una aplicación web para gestionar tareas académicas.

---

# Tecnologías utilizadas

- TypeScript
- Vite
- TailwindCSS
- LocalStorage
- GitHub

---

# Funcionalidades

- Crear tareas
- Editar tareas
- Eliminar tareas
- Filtrar tareas
- Mostrar resumen estadístico
- Persistencia usando LocalStorage

---

# Instalación

## Clonar repositorio

```bash
git clone URL_DEL_REPOSITORIO
```

## Entrar al frontend

```bash
cd frontend
```

## Instalar dependencias

```bash
npm install
```

## Ejecutar proyecto

```bash
npm run dev
```
